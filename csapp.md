# 0 全书概览
全书共12章，旨在阐述计算机系统的核心概念。
* 第1章：计算机系统漫游。这一章通过研究“hello, world”这个简单程序的生命周期，介绍计算机系统的主要概念和主题。
* 第2章：信息的表示和处理。讲述了计算机的算术运算，重点描述了会对程序员有影响的无符号数和数的补码表示的特性。
* 第3章：程序的机器级表示。教读者如何阅读由C编译器生成的x86-64机器代码。
